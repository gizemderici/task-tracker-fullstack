Task Tracker Full Stack
Proje Hakkında

Bu proje, staj öncesinde NextJS, NestJS ve PostgreSQL teknolojilerini öğrenmek amacıyla geliştirilmiş bir görev takip uygulamasıdır.

Kullanıcılar görev ekleyebilir, tamamlayabilir, silebilir ve görevlerini filtreleyebilir.
