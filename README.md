\# Task Tracker Full Stack



\## Proje Hakkında



Bu proje NextJS, NestJS ve PostgreSQL kullanılarak geliştirilmiş bir görev takip uygulamasıdır.



Kullanıcılar görev ekleyebilir, silebilir, tamamlayabilir ve görevlerini filtreleyebilir.



\## Kullanılan Teknolojiler



\### Frontend



\* NextJS

\* TypeScript

\* Tailwind CSS



\### Backend



\* NestJS

\* Prisma ORM



\### Database



\* PostgreSQL

\* Docker



\## Özellikler



\* Görev ekleme

\* Görev silme

\* Görev tamamlama

\* Öncelik belirleme

\* Görev filtreleme

\* Görev arama



\## Kurulum



\### Backend



```bash

cd backend

npm install

npm run start:dev

```



\### Frontend



```bash

cd frontend

npm install

npm run dev

```
