\# 🚀 Task Tracker Full Stack



Modern bir görev takip uygulaması.



Bu proje, \*\*NextJS\*\*, \*\*NestJS\*\* ve \*\*PostgreSQL\*\* kullanılarak geliştirilmiş bir Full Stack uygulamadır.



\---



\## 📌 Proje Hakkında



Kullanıcılar:



\* ➕ Görev ekleyebilir

\* ✅ Görev tamamlayabilir

\* 🗑️ Görev silebilir

\* 🎯 Öncelik belirleyebilir

\* 🔍 Görev arayabilir

\* 📂 Görev filtreleyebilir



\---



\## 🛠️ Kullanılan Teknolojiler



\### Frontend



\* NextJS

\* TypeScript

\* Tailwind CSS



\### Backend



\* NestJS

\* Prisma ORM



\### Database



\* PostgreSQL

\* Docker

\## ⚙️ Kurulum



\### Backend



```bash

npm install

npm run start:dev

```



\### Frontend



```bash

cd frontend

npm install

npm run dev

```
